# Academy Cinemas – Movie Theatre Website

A modern, responsive cinema website featuring classic films, interactive UI components, and a fully functional ticket-selection system. Built using Bootstrap 5, custom CSS, and JavaScript enhancements for smooth user interaction.

## Features

- Responsive Navbar with dropdowns for “Now Playing”

- Movie Sections with posters, descriptions, and showtimes

- Bootstrap Popovers showing dynamic star ratings on hover

### Ticket Booking System:

- Choose movie

- Select time

- Pick ticket quantity

- “Buy Tickets” button with loading animation

- Bootstrap Toast displaying purchase confirmation

- Contact Form with clean, Bootstrap-styled inputs

- Navbar Shrink Effect on scroll via jQuery

- Fully responsive layout (custom media CSS included)

## Tech Stack

- HTML5

- Bootstrap 5

- CSS3 (custom + media queries)

- JavaScript ES6

- jQuery for scroll + menu behavior

## UI Interactions

- Hover popovers with rating images

- Auto-collapsing mobile menu

- Loading spinner for ticket purchases

- Toast notifications for confirmation
